# 📋 كشف الطالب — تطبيق PWA للدراسات الإسلامية

## 🚀 طريقة التثبيت على Android

### الطريقة 1: عبر Chrome (الأسهل)
1. انسخ مجلد `gradebook-pwa` كاملاً إلى استضافة ويب أو خادم محلي
2. افتح الرابط في متصفح **Chrome** على الجهاز
3. اضغط على قائمة Chrome (⋮) واختر **"إضافة إلى الشاشة الرئيسية"**
4. سيظهر التطبيق كأيقونة مستقلة على الهاتف

### الطريقة 2: فتح محلي بدون استضافة
- يمكن فتح `index.html` مباشرة في المتصفح من الجهاز
- لكن Service Worker يحتاج HTTPS أو localhost للعمل

### الطريقة 3: خادم محلي Python (للتجربة)
```bash
cd gradebook-pwa
python3 -m http.server 8080
# ثم افتح: http://localhost:8080
```
